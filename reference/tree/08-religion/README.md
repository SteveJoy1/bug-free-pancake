## Sourcing notes

- Religious traditions describe themselves in insider terms; scholarly religious studies describes them in outsider terms. This reference uses outsider/academic framings.
- Claims that are doctrinally contested within a tradition are marked as "held by [group]" rather than as facts about the religion.
- Sources: Oxford Handbook of World Religions, Pew Research religion data, standard academic religious-studies surveys.
