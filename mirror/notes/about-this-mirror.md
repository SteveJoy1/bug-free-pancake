# About this Mirror

## Summary

A personal knowledge base: short markdown notes, tagged, with explicit confidence and gap tracking. Two purposes — a reference for things I'd otherwise forget, and a mirror that makes my own blind spots visible.

## Details

- Notes live flat in `notes/`. Categorization is via tags, not folders.
- Every note declares a `confidence` (1–5). Low-confidence notes mark the learning edge — grep for `confidence: 1` or `confidence: 2` to see what to study.
- Every note can declare `gaps` — specific open questions. These are meant to roll up into `GAPS.md` (manual for now, automated later).
- Cross-reference with `[[wiki-links]]`. The graph that emerges is itself a map of what's known.
- Source-only today. A static site can be layered on later without rewriting any notes.
