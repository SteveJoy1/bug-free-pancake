# Tags

Controlled vocabulary. Add a tag here before using it in a note — this keeps the tag space small enough to be useful.

## Format

`tag-name` — one-line description of what belongs here.

## Current tags

- `meta` — notes about this system itself (conventions, how-tos for using the mirror).
- `physics` — physics, broadly.
- `classical-mechanics` — Newtonian/Lagrangian/Hamiltonian mechanics; Propaedia Part 01.
